# Supply Chain & Logistics Dashboard

![Dashboard ](/Supply_Chain_&_Logistics_Dashboard/images/project_image_1.png)

## Indroduction

This project presents an interactive Supply Chain & Logistics Dashboard built using Power BI, designed to provide a consolidated view of end-to-end supply chain performance.

The dashboard enables business users to monitor **revenue, profitability, inventory levels, supplier efficiency, shipment performance, and customer delivery** metrics from a single page, supporting faster and more informed operational and strategic decisions.
